enum FUDaeProfileType::Type

Enumerates the types of effect profiles.

Enumerator:

The HLSL profile for DirectX.

GLSL 

The GLSL profile for OpenGL.

GLES 

The GLES profile for OpenGL ES.

The common profile which encapsulates the standard lighting equations: Lambert, Phong, Blinn.
