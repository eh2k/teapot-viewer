enum FCDEffectParameterSurfaceInitFactory::InitType

The supported initialization types.

Enumerator:
FROM 

Loads a surface from one simple image file.

See also:
FCDEffectParameterSurfaceInitFrom
AS_NULL 

No initialization.

This surface may be initialized by some future effect parameter override.

AS_TARGET 

Initializes an engine-specific render target for offscreen rendering.

In this case, the dimensions should be provided by the surface effect parameter.

CUBE 

Loads a cube-map from one complex image file or six simple image files.

See also:
FCDEffectParameterSurfaceInitCube
VOLUME 

Loads a 3D images for one image file.

See also:

PLANAR 

Loads a surface from one simple image file.
