enum FUDaePassStateBlendEquation::Equation

A render state blend equation formula.

Enumerator:
ADD 

Will add the destination color with the source color.

SUBTRACT 

Will substract the source color to the destination color.

REVERSE_SUBTRACT 

Will substract the destination color to the source color.

MIN 

Will take the smaller of the destination color and the source color components.

MAX 

Will take the larger of the destination color and the source color components.

INVALID 

An unknown blend equation.
